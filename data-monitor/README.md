data-monitor
============
version: 0.0.1

**HOW TO USE**

* Plug-in your Leap Motion

* Open your system terminal
    
    ```
    python dataMonitor.py
    ```
    
**HOW TO CUSTOMIZE**

* Modify **monitor** which is a object of PrettyTable, add rows into it.

* Modify **arch_dir** and **leap_dir** while your dir of libraries different.
